# 🧾 Invoice Generator

A web-based invoice generator built with React, TypeScript, and Vite. Fill in your business details, client info, and line items to instantly generate a printable invoice.

## Features

- Dynamic line item entry with automatic totals
- Print-ready invoice layout
- Fast dev experience via Vite HMR
- Fully typed with TypeScript

## Roadmap

- [ ] Multiple invoice template styles

## Tech Stack

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) with [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) (Oxc)

## Getting Started

git clone https://github.com/Aly26ssa/invoice-generator.git
cd invoice-generator
npm install
npm run dev
